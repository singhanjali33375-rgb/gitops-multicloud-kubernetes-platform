# Flask Microservice Application


## Overview

This is a simple Flask-based microservice created for the GitOps Multicloud Kubernetes Platform project.

The application is containerized using Docker and designed to run on Kubernetes.


## Tech Stack

- Python 3.12
- Flask
- Gunicorn
- Docker
- Kubernetes
